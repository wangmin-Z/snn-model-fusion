# MFT 核心实现

本目录实现论文方法本身，各文件职责如下。

| 文件 | 完成的任务 |
| --- | --- |
| `bmif.py` | 实现 BM-IF 双侧多强度脉冲、膜电位复位和替代梯度。 |
| `models.py` | 构建 ANN ResNet、S-ResNet18/S-ResNet34，并复制匹配的 ANN 权重。 |
| `data.py` | 读取 ImageFolder 数据，执行增强以及 80/20、50/50 分层划分。 |
| `__init__.py` | 暴露常用模型和神经元构建接口。 |

本目录只负责可复用算法，不包含命令行参数、实验队列或训练结果。
